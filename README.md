# VaultKeeper - Secure Password & Document Manager

Professional password manager with military-grade encryption.

## Versions

### Free Version
- Up to 10 passwords
- Up to 5 documents
- Encrypted storage
- Simple interface

### Pro Version ($10/6 months)
- Unlimited passwords & documents
- Password generator
- Strength indicator
- 6 premium themes
- Backup & restore
- Categories & tags
- Priority support
