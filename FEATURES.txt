TIPS FOR BEST USE
==================

1. CREATE STRONG PASSWORDS
   - Use the password generator
   - Aim for "Very Strong" rating
   - Mix all character types
   - Use at least 16 characters

2. ORGANIZE WITH CATEGORIES
   - Use consistent category names
   - Examples: Work, Personal, Banking, Social
   - Makes searching easier

3. BACKUP REGULARLY
   - Click Backup button weekly
   - Save backups to USB drive or cloud
   - Keep backups in a safe location
   - NEVER lose your secret.key!

4. USE USERNAMES
   - Add email/username for each password
   - Makes it easier to remember which account
   - Helps when logging in

5. ADD NOTES
   - Security questions
   - Account recovery info
   - Special instructions
   - Anything else important
