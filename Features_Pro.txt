  3. Create regular backups
  4. Enjoy peace of mind!

                    üîí SECURITY SPECIFICATIONS
--------------------------------------------------------------------------------

ENCRYPTION
  Algorithm: Fernet (AES-256-CBC + HMAC-SHA256)
  Key Size: 256-bit
  Mode: CBC (Cipher Block Chaining)
  Authentication: HMAC-SHA256

PASSWORD HASHING
  Algorithm: PBKDF2-HMAC-SHA256
  Iterations: 100,000
  Salt: 64 hex characters (32 bytes, unique per install)
  Output: 64 hex characters (32 bytes)

TWO-FACTOR AUTHENTICATION
  Standard: TOTP (RFC 6238)
  Algorithm: HMAC-SHA1
  Time Step: 30 seconds
  Code Length: 6 digits
  Window: ¬±1 step (tolerance for clock drift)

DATA STORAGE
  Auth Files: %APPDATA%\VaultKeeper\ (hidden)
  Vault Files: Application directory
  All Passwords: Encrypted with AES-256
  All Hashes: Salted with unique salt
  No Plain-Text: Ever stored

                    üéØ PRO TIPS
--------------------------------------------------------------------------------

1. ENABLE 2FA
   While optional, 2FA adds an extra layer of security.
   It's FREE to use with any authenticator app!

2. USE STRONG MASTER PASSWORD
   Your master password is the key to everything.
   Use 12-16+ characters with mixed character types.

3. BACKUP REGULARLY
   Create weekly backups and store in multiple locations.
   Test your backups by restoring occasionally.

4. GENERATE STRONG PASSWORDS
   Use the built-in generator for all new passwords.
   Aim for 16+ characters with all character types.

5. ORGANIZE WITH CATEGORIES
   Use categories like "Work", "Personal", "Banking", etc.
   Makes searching and managing passwords easier.

6. STORE IMPORTANT DOCUMENTS
   Use document storage for:
   - License keys
   - Recovery codes
   - Important PDFs
   - Secure notes

7. LOCK YOUR COMPUTER
   Always lock (Win+L) when stepping away.
   VaultKeeper requires password on every launch, but
   locking adds another layer of protection.

8. KEEP LICENSE KEY SAFE
   Store your license key email safely.
   You'll need it for renewal or reinstallation.
